## Objective

Build the Reserve Health Widget — the financial heartbeat of the ShieldGuard insurer dashboard — with all four visual layers (Global Banner, Metric Cards, City Cards, Zone Rows) plus Circuit Breaker controls. Assemble into the main dashboard page.
