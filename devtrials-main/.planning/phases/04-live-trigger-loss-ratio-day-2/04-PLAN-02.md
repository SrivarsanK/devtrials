# Plan 04-02: Loss Ratio Area Chart

## Objective

Implement a production-grade 30-day loss ratio area chart using the D3-for-math/React-for-DOM pattern. The chart features threshold-based gradient fills that visually communicate risk zones (green < 65% < amber < 80% < red) and an interactive hover tooltip for daily drill-down.

`'use client'` component with the following structure:

```typescript
'use client';

import { useMemo, useState, useRef, useCallback } from 'react';
import * as d3 from 'd3';
import { useLossRatio } from '@/hooks/useLossRatio';
import { LOSS_RATIO_GREEN, LOSS_RATIO_AMBER } from '@/constants/thresholds';
import { LossRatioDataPoint } from '@/types';
```

**Chart dimensions:**
```typescript
const MARGIN = { top: 20, right: 30, bottom: 40, left: 50 };
const WIDTH = 600;
const HEIGHT = 320;
const innerWidth = WIDTH - MARGIN.left - MARGIN.right;
const innerHeight = HEIGHT - MARGIN.top - MARGIN.bottom;
```

**D3 scales (useMemo):**
```typescript
const xScale = useMemo(() =>
  d3.scaleTime()
    .domain(d3.extent(data, d => new Date(d.date)) as [Date, Date])
    .range([0, innerWidth]),
  [data, innerWidth]
);

const yScale = useMemo(() =>
  d3.scaleLinear()
    .domain([0, 100])
    .range([innerHeight, 0]),
  [innerHeight]
);
```

**Area and line generators (useMemo):**
```typescript
const areaGenerator = useMemo(() =>
  d3.area<LossRatioDataPoint>()
    .x(d => xScale(new Date(d.date)))
    .y0(innerHeight)
    .y1(d => yScale(d.lossRatioPercent))
    .curve(d3.curveMonotoneX),
  [xScale, yScale, innerHeight]
);

const lineGenerator = useMemo(() =>
  d3.line<LossRatioDataPoint>()
    .x(d => xScale(new Date(d.date)))
    .y(d => yScale(d.lossRatioPercent))
    .curve(d3.curveMonotoneX),
  [xScale, yScale]
);
```

**Compute gradient stops — this is the key visual feature (LRC-003):**
The `<linearGradient>` must be VERTICAL (y1=0, y2=1) with stops at the y-positions corresponding to 65% and 80% on the yScale:
```typescript
const gradientStops = useMemo(() => {
  const y65 = yScale(LOSS_RATIO_GREEN) / innerHeight; // fraction from top
  const y80 = yScale(LOSS_RATIO_AMBER) / innerHeight;
  return [
    { offset: '0%', color: '#EF4444', opacity: 0.3 },        // top = 100% ratio = red
    { offset: `${y80 * 100}%`, color: '#EF4444', opacity: 0.3 },
    { offset: `${y80 * 100}%`, color: '#F59E0B', opacity: 0.25 },
    { offset: `${y65 * 100}%`, color: '#F59E0B', opacity: 0.25 },
    { offset: `${y65 * 100}%`, color: '#10B981', opacity: 0.2 },
    { offset: '100%', color: '#10B981', opacity: 0.2 },       // bottom = 0% ratio = green
  ];
}, [yScale, innerHeight]);
```

**X-axis ticks:** Compute ~6 date ticks using `xScale.ticks(6)`, format as "MMM DD" using `d3.timeFormat('%b %d')`.

**Y-axis ticks:** Use `[0, 20, 40, 60, 80, 100]` with "%" suffix.

**Hover state:**
```typescript
const [hoveredIndex, setHoveredIndex] = useState<number | null>(null);
const svgRef = useRef<SVGSVGElement>(null);
```

**Mouse move handler (useCallback):**
```typescript
const handleMouseMove = useCallback((e: React.MouseEvent<SVGSVGElement>) => {
  if (!svgRef.current || data.length === 0) return;
  const svgRect = svgRef.current.getBoundingClientRect();
  const mouseX = e.clientX - svgRect.left - MARGIN.left;
  const bisector = d3.bisector((d: LossRatioDataPoint) => new Date(d.date)).left;
  const date = xScale.invert(mouseX);
  const index = bisector(data, date, 1);
  const nearest = Math.max(0, Math.min(data.length - 1,
    index > 0 && (date.getTime() - new Date(data[index - 1].date).getTime()) <
    (new Date(data[index]?.date ?? data[index - 1].date).getTime() - date.getTime()) ? index - 1 : index
  ));
  setHoveredIndex(nearest);
}, [data, xScale]);
```

**SVG rendering structure:**
```tsx
<svg ref={svgRef} viewBox={`0 0 ${WIDTH} ${HEIGHT}`} className="w-full h-auto" onMouseMove={handleMouseMove} onMouseLeave={() => setHoveredIndex(null)}>
  <defs>
    <linearGradient id="lossRatioGradient" x1="0" y1="0" x2="0" y2="1">
      {gradientStops.map((stop, i) => (
        <stop key={i} offset={stop.offset} stopColor={stop.color} stopOpacity={stop.opacity} />
      ))}
    </linearGradient>
  </defs>
  <g transform={`translate(${MARGIN.left},${MARGIN.top})`}>
    {/* Y grid lines */}
    {/* Y axis labels */}
    {/* X axis labels */}
    {/* Area fill */}
    <path d={areaGenerator(data) || ''} fill="url(#lossRatioGradient)" />
    {/* Line stroke */}
    <path d={lineGenerator(data) || ''} fill="none" stroke="white" strokeWidth={2} strokeOpacity={0.8} />
    {/* 65% threshold dashed line */}
    <line x1={0} x2={innerWidth} y1={yScale(65)} y2={yScale(65)} stroke="#10B981" strokeDasharray="6 4" strokeWidth={1} />
    <text x={innerWidth + 5} y={yScale(65) + 4} fill="#10B981" fontSize={10}>Target (65%)</text>
    {/* 80% threshold dashed line */}
    <line x1={0} x2={innerWidth} y1={yScale(80)} y2={yScale(80)} stroke="#EF4444" strokeDasharray="6 4" strokeWidth={1} />
    <text x={innerWidth + 5} y={yScale(80) + 4} fill="#EF4444" fontSize={10}>Alert (80%)</text>
    {/* Hover crosshair + tooltip */}
    {hoveredIndex !== null && data[hoveredIndex] && (
      <>
        <line x1={xScale(new Date(data[hoveredIndex].date))} x2={xScale(new Date(data[hoveredIndex].date))} y1={0} y2={innerHeight} stroke="white" strokeOpacity={0.3} strokeDasharray="3 3" />
        <circle cx={xScale(new Date(data[hoveredIndex].date))} cy={yScale(data[hoveredIndex].lossRatioPercent)} r={4} fill="white" />
      </>
    )}
  </g>
</svg>
```

**Tooltip (positioned outside SVG, rendered as a div overlay):**
```tsx
{hoveredIndex !== null && data[hoveredIndex] && (
  <div className="absolute bg-shield-card border border-shield-border rounded-lg p-3 shadow-xl pointer-events-none text-sm" style={{ left: tooltipLeft, top: tooltipTop }}>
    <p className="text-shield-muted text-xs">{data[hoveredIndex].date}</p>
    <p className="text-white font-bold">Loss Ratio: {data[hoveredIndex].lossRatioPercent}%</p>
    <p className="text-shield-green">Premiums: ₹{data[hoveredIndex].premiumsCollectedLakhs} L</p>
    <p className="text-shield-amber">Claims: ₹{data[hoveredIndex].claimsPaidLakhs} L</p>
  </div>
)}
```

Wrap the whole component in a `<div className="relative">` to position the tooltip absolutely.

## Key Decisions
- D3 for math only, React for DOM — ensures clean component lifecycle and avoids double-render issues
- viewBox-based SVG sizing makes the chart responsive without JS resize handling
- Gradient stops computed from yScale positions so they always align with the data domain
- Tooltip rendered as HTML div overlay (not SVG foreignObject) for better styling control
