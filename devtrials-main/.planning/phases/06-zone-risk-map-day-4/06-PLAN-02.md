# Plan 06-02: Zustand Reactivity + Dashboard Integration

## Objective

The `LeafletMapInner` already subscribes to Zustand (reactivity is built in via `useEffect` deps on `reserveHealth` and `pausedZones`). This plan focuses on:
1. Injecting custom CSS to make Leaflet popups dark-themed.
2. Wiring `ZoneRiskMap` into `DashboardClient`.
